Prime Chains

Let's take a number sequence
of length N.

If the sum of all the numbers except the last is evenly divisible by that last number (or the Nth number), we'll call the sequence a "Clean Chain of Length N" .

This algorithm finds "Clean Chains"
consisting of only consecutive prime numbers.

The first chain is simply an array consisting of {2}. Because two is the only number in the list, it is also the last number in the list, or the Nth number. The sum of every digit in the list except for N is then zero. Two divided by zero leaves no remainder, which leaves us with our first "Clean Chain".

The second chain is {2, 3, 5} because (2 + 3) / 5 has no remainder

This code finds the first four chains
only because a big number javascript library wold be necessary to make the necessary calculations for the fifth chain.
var primer = primeArray[primeArray.length - 1];
  var instanceGen = function() {
   var z = 0;
   do {
// Find primes until (primeSum % last prime) == 0
    primer += 1;
    var notPrime = false;
    var n = 0
    do {
// Do/while loop closes when a prime is found
     if (z == 0) { // For small chains
      z++ ;
      primer++;
     } else if (primer % primeArray[n] == 0) {
// If primer is evenly divisible by a number...
      notPrime = true;
     } else {
// When a prime number is found...
      n+=1;
      if (primeArray[n] > Math.sqrt(primer)) {
       primeSum += primeArray[primeArray.length-1];
       primeArray.push(primer);
// Ensures do/while loop closes
       notPrime = true        }
    }
   } while (notPrime == false)
  } while (primeSum % primeArray[primeArray.length - 1] != 0);
}
