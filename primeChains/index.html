Let's take a number sequence
of length N.

If the sum of the first N - 1 numbers is evenly divisible by the final number (or Nth number), we'll call the sequence a "Clean Chain of Length N" .

This algorithm finds "Clean Chains"
consisting of only consecutive prime numbers.

The first chain is simply an array consisting of {2}. Because N = 1 (the chain's length is one), N - 1 = 0. Two, the Nth number, is evenly divisble into zero, which leaves us with our first "Clean Chain".

The second chain is {2, 3, 5} because (2 + 3) / 5 has no remainder

This code finds the first four chains
only because a big number javascript library wold be necessary to make the necessary calculations for the fifth chain.
